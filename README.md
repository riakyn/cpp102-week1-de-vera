## Reflection 
During our first week, I learned that Git and GitHub are important tools for programming students like me. Git helps keep track of changes in our code, allowing us to go back to a previous version whenever we need to fix mistakes or make revisions instead of starting from scratch. On the other hand, GitHub is useful because it allows us to store our code online and share our projects with others.
